# Kazan Music Format

Рабочая папка для переработки статьи «Музыкальная жизнь Казани» в хронологический интерактивный формат.

## С чего начинать работу

Если продолжаем этот проект после паузы, сначала читать:
1. `page-structure-main.md` — основная структура страницы
2. `writing-rules.md` — обязательные правила сборки текста
3. `text-draft.md` — текущий черновик текста

Во вторую очередь можно обращаться к `chapter-philosophy.md` — это вспомогательный файл с философией блоков и внутренней логикой глав.

## Что здесь хранится

- структура страницы
- черновики коротких блоков
- развороты по блокам
- заметки по фактуре и логике сборки
